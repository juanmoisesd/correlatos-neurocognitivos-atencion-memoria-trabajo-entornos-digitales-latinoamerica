# Codebook

This codebook describes the variables in the dataset.

## Variables
1. `subject_id`: Unique identifier for each participant.
2. `age`: Age of the participant in years.
3. `gender`: Gender identity (Male/Female/Other).
4. `digital_hours`: Number of hours spent in digital environments per day.
5. `attention_score`: Score on the attention assessment (0-100).
6. `wm_score`: Score on the working memory assessment (0-100).
7. `region`: Geographical region in Latin America.
