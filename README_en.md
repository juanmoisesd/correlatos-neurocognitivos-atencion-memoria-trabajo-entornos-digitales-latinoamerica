# Neurocognitive Correlates of Attention and Working Memory in Digital Environments in Latin America

## Description
Database of figures and statistics on the neurocognitive correlates of attention and working memory in digital environments in Latin America. Includes data on cognitive performance, use of digital technology, and its impact on attentional and working memory processes.
